# Splunk Home Lab

**Objective:** Simulate security events and analyze logs using Splunk.

**Environment:**
Oracle Virtual Box:
- Windows 11 VM: Event logs, user login events
- Kali Linux VM: Syslogs, simulated attacks (nmap scans, sudo commands)

**Tools:** Splunk Enterprise, GitHub, Windows Event Viewer, Kali Linux tools

**Steps Taken:**
1. Installed Splunk on Windows 10 VM
2. Configured log collection from Windows and Linux
3. Generated events: failed login attempts, network scans
4. Analyzed logs with SPL
5. Created dashboards and alerts
